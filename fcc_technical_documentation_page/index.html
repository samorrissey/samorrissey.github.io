Variables

A variable is a container for storing information. The variable is an important step in coding because with variables we can start using an important programming tool, repetition. Variables allow us to quickly reuse a value in our program as well an change a value in our program.

In ES6, let and const are the preferred variable keywords for many programmers. var is the pre-ES6 keyword for variable.

Declaring a Variable Using let

let myName = "Sara";

The above example can be broken down into its basic parts.

  • let is a variable keyword. The JavaScript interpreter knows this keyword is used to create a variable.
  • myName is the the variable name. In order to use a variable you must give it a name. This is sometimes called the identifier.
  • = is referred to as the assignment operator; this tells the interpreter that you are going to assign a value to the variable. It is also used to update the value given to a variable.
  • "Sara" is the variable's value.

Using let, signals to the program that the variable can be reassigned to a different variable.

Declaring a Variable Using const

const myName = "Sara";

const is short for constant. A variable assigned using const cannot be reassigned later in the program. If you try, the program will throw a TypeError. Additionally, const variables must be assigned when they are declared or the program will throw a SyntaxError.

If you do not need to reassign a variable throughout the program, use const otherwise, use let.

Data Types

Data types describe the different types of data that can be stored or manipulated within a program. They tell us about the data, including how it can be stored and what types of operations we can perform. JavaScript has six fundamental data types. They are:

  • Number
  • String
  • Boolean
  • Null: the intentional absence of a value represented by the keyword null
  • Undefined: denoted by the keyword undefined (without quotes) and represents the absence of a value
  • Object: collections of related data

    • There are two categories of data types, primitives and composite. The primitive data types exist in almost every programming language. They are:

    Arrays

    Arrays are JavaScript's way of making lists. An array is a special type of variable that stores a list of values. When creating an array, you do not need to specify how many values it will hold. Arrays can store any data type including:

    • strings
    • numbers
    • booleans

    Creating An Array

    One way to create an array is to create an array literal. It creates an array by wrapping items in square brackets.

    Array Literal Syntax

    ['element example', 10, true]

    The above example can be broken down into its basic parts.

    • The array is represented by the square brackets [] and the content inside
    • Each content item instide an array is called an element
    • There are three elements inside this example array
    • Each element inside this example array is a different data type

    Arrays can be saved to a variable.

    Functions

    A function is a reusable block of code that groups together a sequence of statements to perform a specific task. Functions are used to organize a program into distinct units, making interchanging and editing them easier. This makes the entire programs organized and modular.

    A JavaScript function is defined with the function keyword followed by a name, followed by parentheses. The code to be executed by the function is placed insided curly braces.

    Function Declaration Syntax

    function greetWorld() { console.log('Hello, World!'); }

    The above example can be broken down into its basic parts.

    • function is the function keyword that allows us to declare a function
    • greetWorld() is the function name. This is also called an identifier
    • {console.log('Hello, World!');} is the function's statements, or the code to be executed

    Declaring a function does not ask the code inside the function body to run, it just declares the existence of the function. The code inside a function body executes only when a function is called.

    Calling a Function Syntax

    greetWorld(); // prints 'Hello, World!'

    In order to call a function in code, type the function name followed by parentheses. This function call executes the function body, or all of the statement between the curly braces in the function declaration.

    • Note: You can call the same function as many times as needed.
